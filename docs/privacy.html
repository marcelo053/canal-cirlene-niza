Política de Privacidade — Canal Cirlene Niza

Última atualização: 18 de maio de 2026

1. Introdução

O Canal Cirlene Niza respeita sua privacidade. Esta política descreve como tratamos informações no contexto da nossa plataforma de publicação de conteúdo.

2. Dados que Não Coletamos

Não coletamos, armazenamos ou processamos dados pessoais de espectadores ou usuários das plataformas onde publicamos conteúdo (TikTok, YouTube, Instagram).

3. Integração com TikTok API

Utilizamos a API do TikTok exclusivamente para publicar conteúdo na conta oficial @ciciniza1. Os dados de autenticação são armazenados de forma segura e utilizados somente para essa finalidade. Não acessamos, coletamos ou compartilhamos dados de outros usuários do TikTok.

4. Dados de Autenticação

Tokens de acesso OAuth utilizados para integração com plataformas são armazenados de forma segura e nunca compartilhados com terceiros. São utilizados exclusivamente para publicação automatizada de conteúdo nas contas oficiais do canal.

5. Cookies

Este site não utiliza cookies.

6. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos quaisquer dados com terceiros.

7. Políticas de Terceiros

As plataformas onde publicamos conteúdo possuem suas próprias políticas de privacidade:

  • TikTok Privacy Policy
  • Google / YouTube Privacy Policy
  • Instagram Privacy Policy
